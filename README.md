JavaScript サンプルアプリケーション集
===

リポジトリ分けるのウザいので、JS系のサンプルはここにまとめる。  
作成したサンプルアプリケーションは以下。

- angular-sample
  - Angular CLI使ってみた
- es6-sample
  - webpack + Babel6 でブラウザで動く最低限のアプリ
- express-sample
  - ```express-generator``` 使ってみた
- line-chatbot
  - LINEチャットボットのサンプル
- mean2-sample
  - MEANスタックのAngular（所謂Angular2）バージョン
  - ディレクトリ構成検討と動作確認のために作った
- nodejs-sample
  - NodeJSの最低限の使い方確認用
